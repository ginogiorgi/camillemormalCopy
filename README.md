# Camille Mormal — Horizontal Image Scroll

Recreation of the horizontal drag-to-scroll effect with parallax from [camillemormal.com](https://camillemormal.com/).

🌐 **[Live demo →](https://ginogiorgi.github.io/camillemormalCopy/)**

## How it works

Click and drag horizontally to scroll through images. As you drag, each image shifts its `objectPosition` in sync — creating a parallax depth effect.

## Stack

- Vanilla JavaScript
- HTML + CSS
